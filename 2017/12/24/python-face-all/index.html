发表于 2017-12-24 | 分类于 programming ， python | | 本文总阅读量 次

参考资料

主要代码参考了这个系列的视频，但是这个视频是基于python2的，opencv也不是最新版本，以下代码做了python3更新。
并且添加了谷歌语音api的连接。

beta版完整的代码在我的Github上

目标

  1. 识别出人脸
  2. 按识别出的人名问好

操作环境

  • windows10 ，（Linux 应该也可以，没试过）
  • python3.5.0
  • python 主要用到的库
    • opencv3.3.1 图像处理用
    • gTTS 文字转语音用
    • pygame 播放语音文件
    • threading 多线程处理

注意：python和opencv的版本不一样的话，无法执行。
如何安装python和opencv看这个链接或者自己谷歌。
